Preguntas frecuentes (FAQ)

Puedo utilizar Scratch Link para conectar a extensiones?

S. Sin embargo, necesitars una conexin a Internet para utilizar Scratch Link.

Puedo compartir proyectos desde Scratch para escritorio?

Actualmente no tenemos esta funcionalidad. Por ahora, puedes guardar un proyecto en Scratch para escritorio, subirlo a tu cuenta de Scratch, y compartirlo aqu. En una version posterior agregaremos la funcionalidad de subir proyectos a tu cuenta de Scratch directamente desde la versin de escritorio.

Puedo utilizar Scratch para escritorio y tambin tener Scratch abierto en el navegador?

S.

Cundo estar disponible Scratch para escritorio en Chromebook?

Scratch para escritorio an no est disponible para Chromebook. Estamos trabajando en ello y esperamos que est ms tarde en 2019.

Cundo estar disponible Scratch para escritorio en Linux?

Actualmente Scratch para escritorio no est disponible para Linux. Estamos trabajando con nuestros colaboradores y con la comunidad de software libre para determinar si es posible tener una versin para Linux a futuro. Estn atentos!
